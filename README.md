# 项目简介 java1138

本项目是一个基于 Python + Django 开发的图书管理系统，采用 sqlite 作为数据库，设计上专注于提供便捷的后台管理功能。

## 技术栈

- 后端：Python 3.x
- 框架：Django
- 数据库：SQLite

## 功能模块

- 登陆模块
- 图书管理
- 借阅管理
- 个人管理
- 会员管理
- 数据可视化图表

## 系统角色

本项目为单一后台端，主要面向管理员用户设计。

## 运行环境

需自行配置 Python 和 Django 环境以确保项目正常运行。

## 部署步骤

1. 环境准备：安装 Python 3.x 和 Django。
2. 数据库配置：使用 SQLite。
3. 代码部署：将代码部署到服务器。
4. 配置运行：进行必要的配置后运行项目。
